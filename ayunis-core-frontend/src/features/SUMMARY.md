# Cross-Cutting Logic

Self-contained business logic modules shared across multiple pages and widgets.

Seven feature modules and several standalone hooks provide cross-cutting concerns: theme management (dark/light mode persistence), language switching (German/English via i18next), model/provider configuration with provider icon utilities, usage analytics data hooks for statistics, charts, and distribution metrics, skill action hooks for toggling skill active/pinned state, MCP OAuth authorization controls shared across integration management screens, and MCP user-config management for reading and updating user-level MCP configuration.

The `theme/` feature manages dark/light mode with localStorage persistence through `useTheme`, `setTheme`, and `getTheme` utilities. The `language/` feature wraps i18next language switching in a `useLanguage` hook. The `models/` feature provides `useModelsWithConfig` and `useProviders` hooks for fetching AI model configurations, plus provider constants and icon utilities for Ollama, Mistral, Anthropic, and OpenAI. The `usage/` feature contains analytics hooks (`useUsageStats`, `useUserUsage`, `useProviderUsage`, `useProviderUsageChart`, `useModelDistribution`, `useUsageConfig`) powering admin dashboard charts. The `skill-actions/` feature provides `useToggleSkillActive` and `useToggleSkillPinned` hooks for toggling skill active and pinned state. The `mcp-oauth/` feature encapsulates the shared OAuth authorization flow for MCP integrations with `useStartOAuthAuthorization`, `useRevokeOAuthAuthorization`, `useOAuthStatus`, and the reusable `OAuthAuthorizeButton` and `OAuthStatusBadge` components. The `mcp-user-config/` feature provides `useGetUserConfig` and `useSetUserConfig` hooks for fetching and updating user-level MCP configuration, along with a reusable `UserConfigDialog` component. Standalone hooks at the feature root include `useAgents` for agent listing, `useDeleteThread` and `useRenameThread` for thread management, `useAutoScroll` for chat scrolling behavior, `useKeyboardShortcut` for hotkey bindings, `usePermittedModels` for org-scoped model access, `useReleaseNotes` for changelog display, `useRedirectNotification` for post-navigation toasts, and `useGtm` for Google Tag Manager integration.
