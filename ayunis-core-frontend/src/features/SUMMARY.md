# Cross-Cutting Logic

Self-contained business logic modules shared across multiple pages and widgets.

Seven feature modules and several standalone hooks provide cross-cutting concerns: theme management (dark/light mode persistence), language switching (German/English via i18next), model/provider configuration with provider icon utilities, usage analytics data hooks for statistics, charts, and distribution metrics, skill action hooks for toggling skill active/pinned state, marketplace configuration for exposing marketplace availability and destination URLs, and fair use limit editing for shared limit validation and edit state.

The `theme/` feature manages dark/light mode with localStorage persistence through `useTheme`, `setTheme`, and `getTheme` utilities. The `language/` feature wraps i18next language switching in a `useLanguage` hook. The `models/` feature provides `useLanguageModels`, `useEmbeddingModels`, `useImageGenerationModels`, and `useProviders` hooks for fetching AI model configurations by type, plus provider constants and icon utilities for Ollama, Mistral, Anthropic, and OpenAI. The `usage/` feature contains analytics hooks (`useUsageStats`, `useUserUsage`, `useProviderUsage`, `useProviderUsageChart`, `useModelDistribution`, `useUsageConfig`) powering admin dashboard charts. The `skill-actions/` feature provides `useToggleSkillActive` and `useToggleSkillPinned` hooks for toggling skill active and pinned state. The `marketplace/` feature provides a `useMarketplaceConfig` hook that returns marketplace availability plus the configured destination URL for marketplace entry points. The `fair-use-limit-editor/` feature provides `useFairUseLimitEditor` for reusable fair use limit edit state and validation. Standalone hooks at the feature root include `useAgents` for agent listing, `useDeleteThread` and `useRenameThread` for thread management, `useAutoScroll` for chat scrolling behavior, `useChatsSidebarOpen` for persisting chats sidebar collapsed state, `useKeyboardShortcut` for hotkey bindings, `usePermittedModels` for org-scoped model access, `useReleaseNotes` for changelog display, `useRedirectNotification` for post-navigation toasts, and `useGtm` for Google Tag Manager integration.
